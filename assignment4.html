Assignment 4

Posting a Page to GitHub

This assignment is to upload all the work done so far to the GibHub Pages.

GitHub grants you a online storage and URL per each of your source repository that you can use to host a website for that repository. In order to take advantage of this, you just need to create a branch named 'gh-pages' in the repository you want to attach the website, and upload all your website files to that branch. You can then access the website using the URL 'http://<Your-User-Name>.github.io/<Your-Repo-Name>/'.

This GitHub page contains more detailed instructions on how to get this done. It is fairly simple; I was able to upload this website and get it to show on the browser in less than 10 minutes or so.

The GitHub page URL for this website is http://yongcho.github.io/GUI-Programming-1.
